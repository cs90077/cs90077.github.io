如何取得多行文字的實際高度

Sep 17, 2021 1 分鐘閱讀 iOS 

- 準備一段會顯示多行的文字
- 放到Label中
- 透過文字的長度以及Label的寬度，來計算出實際Label的高度

 1// ViewController.swift
 2
 3func setupView() {
 4    let text = "弟子規全文\n\n總敘\n\n弟子規　聖人訓　首孝弟　次謹信\n汎愛眾　而親仁　有餘力　則學文\n\n入則孝\n\n父母呼　應勿緩　父母命　行勿懶\n父母教　須敬聽　父母責　須順承\n冬則溫　夏則凊　晨則省　昏則定\n出必告　反必面　居有常　業無變\n事雖小　勿擅為　苟擅為　子道虧\n物雖小　勿私藏　苟私藏　親心傷\n親所好　力為具　親所惡　謹為去\n身有傷　貽親憂　德有傷　貽親羞\n親愛我　孝何難　親憎我　孝方賢\n親有過　諫使更　怡吾色　柔吾聲\n諫不入　悅復諫　號泣隨　撻無怨\n親有疾　藥先嘗　晝夜侍　不離床\n喪三年　常悲咽　居處變　酒肉絕\n喪盡禮　祭盡誠　事死者　如事生\n"
 5
 6    lblMessage.numberOfLines = 0
 7    lblMessage.text = text
 8    
 9    let height = getTextHeight(text: txt, weightMax: lblMessage.frame.width)
10    lblHeight.text = "height = \(height)"
11}
12
13func getTextHeight(text:String, weightMax:CGFloat) -> CGFloat {
14    let textFont = UIFont.systemFont(ofSize: 17)
15    let textMaxSize = CGSize(width: weightMax, height: CGFloat(MAXFLOAT))
16    let textLabelSize = text.boundingRect(with: textMaxSize, options: [.usesLineFragmentOrigin], attributes: [NSAttributedString.Key.font : textFont], context: nil).size
17    
18    let height = textLabelSize.height
19    return height
20}
